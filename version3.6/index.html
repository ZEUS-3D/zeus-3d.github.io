What is ZEUS-3D?

ZEUS-3D is a multi-physics magneto- hydrodynamical (MHD) code written in FORTRAN77 designed primarily for, but not restricted to, astrophysical applica- tions. For an overview of its design, capabilities, and recent improvements, see What is ZEUS-3D?. For compre- hensive instructions on how to use the code, see the ZEUS-3D user manual.

A complete description of most of the algorithms used in ZEUS-3D and many of its tests may be found in:

1. Clarke, D., 1996, ApJ, 457, 291.
2. Clarke, D., 2010, ApJS, 187, 119.
